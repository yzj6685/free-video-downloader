# 需求分析

## 当前完成状态

项目已经完成核心闭环：

- 公开视频单链接解析与下载。
- B 站、抖音公开视频兜底解析。
- AI 视频总结、字幕文本、思维导图和视频问答。
- 登录注册。
- 免费账号 3 次 AI 总结额度。
- Stripe Checkout 一次性购买 Pro。
- Stripe webhook 写入 Pro 权益。
- Pro 会员 AI 视频总结不限次数。
- 防止已开通 Pro 的账号重复支付。

## 用户目标

目标用户希望把公开视频链接快速转成可保存的视频资料和可复习的内容笔记。核心诉求包括：

- 粘贴链接即可解析。
- 能选择清晰度和格式。
- 下载动作稳定，不打开空白页。
- 有字幕时能提取字幕。
- 无字幕时可通过 ASR 转写。
- AI 自动总结视频内容。
- 能基于视频内容追问。
- 免费可体验，重度使用可开通会员。

## 功能需求

### 视频解析下载

- 支持 URL 分享文案提取。
- 支持无协议链接自动补 `https://`。
- 支持公开视频信息解析。
- 展示标题、作者、时长、封面或预览、格式、大小。
- 支持同源下载和后端中转下载。
- 对平台限制、登录权限、非视频内容给出明确错误提示。

### AI 总结

- 登录后可使用 AI 总结。
- 免费账号可成功生成 3 次 AI 视频总结。
- Pro 会员不限次数。
- 优先使用平台字幕。
- 无字幕时可使用 SiliconFlow ASR 转写。
- 总结结果包含摘要、大纲、知识点、字幕文本、思维导图和建议问题。
- 支持基于当前分析结果的视频问答。

### 账号与会员

- 支持注册、登录、恢复会话和退出登录。
- 免费次数和 Pro 权益绑定登录账号邮箱。
- AI 接口以后端登录 token 为准，不信任前端传入的邮箱。
- 已开通 Pro 的账号不能重复创建 Stripe 支付。

### 支付

- 使用 Stripe Checkout。
- 使用 Stripe webhook 确认支付成功。
- 本地 SQLite 记录订单和权益。
- 支付成功回跳后刷新当前账号权益。

## 非目标

项目不实现：

- DRM 绕过。
- 破解或绕过会员墙。
- 下载私密内容。
- 公共账号 Cookie 注入。
- 自动盗取用户平台登录态。
- 字幕翻译。
- 团队版会员。
- 批量队列。

## 合规要求

产品只面向公开或用户有权访问的视频内容。页面、文档和错误提示都应强调平台规则和版权要求，不承诺“全平台任意下载”。
